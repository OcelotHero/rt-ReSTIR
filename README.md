

- macOS:
    - go to LunarG Vulkan SDK download page and download the appropriate SDK
    - Install LunarG
    - Set the VULKAN_SDK env var
        ```bash
             export VULKAN_SDK=/Users/your_username/vulkan-sdk/1.3.283.0 # Example path
        ```
    - source /.zshrc or bash

execute cargo run
